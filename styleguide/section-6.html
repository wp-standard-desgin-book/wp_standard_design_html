6 コンテンツ

投稿記事内のコンテンツのガイドです

6.1 見出し

投稿記事内の見出しスタイル定義です。

.entry-content 要素の子要素の場合に 各見出しタグにスタイルが適用されます。

見出し - h1

見出し - h2

見出し - h3

見出し - h4

見出し - h5
見出し - h6
<div class="entry-content">
  <h1>見出し - h1</h1>
  <h2>見出し - h2</h2>
  <h3>見出し - h3</h3>
  <h4>見出し - h4</h4>
  <h5>見出し - h5</h5>
  <h6>見出し - h6</h6>
</div>

6.2 テキストのスタイル

投稿記事内のテキストのスタイル定義です。
基本的に、class をつけずにHTMLタグを利用し、 .entry-content 内のスタイル定義をしています。

段落

文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章

small タグ

少し小さなテキストです。

strong タグ

強調テキストです。

定義リスト

定義リスト
定義リストコンテンツ

pre タグ

    .entry-content{
      pre{
        background: $gray;
        padding: 16px;
      }
    }

code タグ

<a href="" title=""> <abbr title=""> <acronym title=""> <b> <blockquote cite=""> <cite> <code> <del datetime=""> <em> <i> <q cite=""> <strike> <strong> 
<div class="entry-content">
  <h2>段落 </h2>
  <p>文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章文章</p>

  <p><small>少し小さなテキストです。</small></p>
  <h2>strong タグ</h2>

  <h2>定義リスト</h2>

    <dt>定義リスト</dt>
    <dd>定義リストコンテンツ</dd>
  </dl>

    .entry-content{

  </code>
</div>

6.5 投稿メタデータ

投稿の著者情報、投稿日、カテゴリリンクのためのメタデータです。

投稿者 : wp-standard-design-book 2014年9月22日 新着情報 
<div class="entry-meta">
	<span class="author">
		投稿者 : wp-standard-design-book
	</span>
	<span class="date">
		<time class="entry-date" datetime="2014-09-22T04:29:32+00:00">2014年9月22日</time>
	</span>
	<span class="categories-links info">
		<a href="http://test.wordpress.com/?cat=1" rel="category">新着情報</a>
	</span>
</div>

6.6 投稿ナビゲーション

「次の記事へ」「前の記事へ」のリンクのためのナビゲーションです。

投稿ナビゲーション

← 前の記事へ 次の記事へ→ 
<nav class="navigation post-navigation" role="navigation">
	<h1 class="screen-reader-text">投稿ナビゲーション</h1>
	<div class="nav-links">
		<a href="/" rel="prev"><span class="meta-nav">←</span> 前の記事へ</a>
		<a href="/" rel="next">次の記事へ<span class="meta-nav">→</span></a>
	</div>
</nav>

6.7 投稿コメントフォーム

投稿のコメントフォームのためのスタイルシートです。

コメントを残す コメントをキャンセル

メールアドレスが公開されることはありません。 *が付いている欄は必須項目です

次の HTMLタグと属性が使えます: <a href="" title=""> <abbr title=""> <acronym title=""> <b> <blockquote cite=""> <cite> <code> <del datetime=""> <em> <i> <q cite=""> <strike> <strong> 

<div id="comments" class="comments-area">
	<div id="respond" class="comment-respond">
		<h3 id="reply-title" class="comment-reply-title">コメントを残す
			<small><a rel="nofollow" id="cancel-comment-reply-link" href="/?p=1#respond" style="display:none;">コメントをキャンセル</a>
			</small>
		</h3>
		<form action="?" method="post" id="commentform" class="comment-form" novalidate="">
			<p class="comment-notes">メールアドレスが公開されることはありません。
				<span class="required">*</span>が付いている欄は必須項目です</p>
			<p class="comment-form-author">
				<label for="author">名前
						<span class="required">*</span>
				</label>
				<input id="author" name="author" type="text" value="" size="30" aria-required="true">
			</p>
			<p class="comment-form-email">
				<label for="email">メールアドレス
						<span class="required">*</span>
				</label>
				<input id="email" name="email" type="email" value="" size="30" aria-required="true">
			</p>
			<p class="comment-form-url">
				<label for="url">ウェブサイト</label>
				<input id="url" name="url" type="url" value="" size="30">
			</p>
			<p class="comment-form-comment">
				<label for="comment">コメント</label>
				<textarea id="comment" name="comment" cols="45" rows="8" aria-required="true"></textarea>
			</p>
			<p class="form-allowed-tags">次の
				<abbr title="HyperText Markup Language">HTML</abbr>タグと属性が使えます:
				<code><a href="" title=""> <abbr title=""> <acronym title=""> <b> <blockquote cite=""> <cite> <code> <del datetime=""> <em> <i> <q cite=""> <strike> <strong></code>
			</p>
			<p class="form-submit">
				<input name="submit" type="submit" id="submit" value="コメントを送信">
				<input type="hidden" name="comment_post_ID" value="1" id="comment_post_ID">
				<input type="hidden" name="comment_parent" id="comment_parent" value="0">
			</p>
		</form>
	</div>
</div>
